# Retime-Re-encode-Tool
Similar to Gmzorz prorec tool (https://github.com/gmzorz/prerecs)
But this tool can also re-time your clips (i.e. 60 fps clips recorded at 0.1 timescale converts into 600fps)

it re-encodes clips to .mp4 optimized for smooth editing in SV/AE

and It supports the following input file types:
.mp4
.avi
.mov
.m4v
